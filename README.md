# Quiz Master Pro

## 📌 About

**Quiz Master Pro** is a console-based interactive quiz game developed in C++ for the Programming Fundamentals course. It demonstrates core programming concepts through an engaging quiz experience with administrative capabilities.

---

## ✨ Features

- Three topics: Riddles, Historical Figures & Animal World
- Three difficulty levels: Easy, Medium & Difficult
- Two question types: True/False & Multiple Choice
- Admin CRUD operations: Add, View, Update & Delete questions
- Password-protected login (Password: 23236)
- Real-time scoring with immediate feedback

---

## 🛠️ Tech Stack

- C++
- Loops (for, while, do-while)
- Conditional Statements (if-else, switch-case)
- Functions (Modular Programming)
- Arrays

---

## 🚀 Quick Start

```bash
g++ main.cpp -o QuizMasterPro
./QuizMasterPro
```
